# api_test
That repo is dedicated to test an api.

# Requirements

Node.js

Express.js

VS Code

Postman/Insomnia

# First Steps

1. Install Node.js dependences using the following commands on Terminal:

* `npm init -y`

* `npm install express`

* `node .`

2. For a better praticity and produtivity, install **nodemon** using the next commands on Terminal too:

* `npm install -g  nodemon`

* `nodemon index.js`

3. For testing the API, you can use Postman/Insomnia (and another option is to install the Thunder Client extension on VS Code).

* After start the API with `nodemon index.js`, you must copy the localhost addres like *https://localhost:8080*.

* Now open Postman/Insomnia or the Thunder Client extension. Paste the localhost address and procide with the tests like the following video.
